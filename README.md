## 🌐 The Problem & The Solution
In today's fast-paced engineering world, **context is the ultimate bottleneck**. New developers spend weeks understanding legacy code, and seasoned architects struggle to map the evolution of massive repositories.

**RepoMind** is your cognitive layer for GitHub. It doesn't just read code; it *thinks* across your repository, providing instant architectural insights, knowledge graphs, and an interactive chat interface that knows your codebase better than its authors.

## ✨ Key Features

- 📁 **Instant Indexing**: Connect any public GitHub repository and watch RepoMind map the entire file structure in seconds.
- 💬 **Codebase Chat**: A persistent, context-aware interface powered by Gemini 1.5 Flash. Ask about bugs, design patterns, or "How does the auth flow work?".
- 🗺️ **Architectural Blueprints**: Automatically generates high-level summaries and structural overviews of your project.
- 🕸️ **Knowledge Graph**: Visualize relationships between files and concepts, making complex dependencies easy to grasp.
- ⏳ **Evolution Timeline**: Track how your project has grown and changed over time.
- 📓 **Smart Notebooks**: Organize your insights into dedicated project notebooks with zero-latency local storage persistence.

## 🛠️ Built With

*   **Frontend**: React.js with TypeScript & Vite for lightning-fast performance.
*   **Intelligence**: Google Gemini 1.5 Flash (via API) for state-of-the-art code analysis.
*   **Animations**: Framer Motion for a sophisticated, executive UI feel.
*   **Styling**: Tailwind CSS with custom glassmorphism and neoclassical design.
*   **Storage**: Intelligent Local Storage persistence (Privacy-focused: your index stays in your browser).
*   **Backend**: Node.js/Express for secure GitHub API orchestration.

## 🚀 Getting Started


### Installation

1.  **Clone & Install**:
    ```bash
    npm install
    ```

2.  **Environment Setup**:
    Create a `.env` file in the root and add your Gemini API Key:
    ```env
    VITE_GEMINI_API_KEY=your_api_key_here
    ```

3.  **Launch the App**:
    ```bash
    npm run dev
    ```

## 📖 Usage

1.  **Add Repository**: Paste any public GitHub URL (e.g., `https://github.com/facebook/react`).
2.  **Explore**: Use the **File Explorer** to navigate or the **Knowledge Graph** to visualize.
3.  **Investigate**: Use the **Chat Interface** to dive deep. Try: *"Explain the main layout hierarchy"* or *"Where are the critical security patterns?"*.
4.  **Manage**: Delete or update notebooks directly from your **Dashboard**.
